Basic Morse Code Translator

How to use this page

Type in input, starting with either English character(a-z, 0-9), or Morse (. or -) for the application to determine type of translation required.

This application only supports the following:

For English To Morse:

Numbers: 0-9 Letters: A-Z Spaces: normal spaces

For Morse To English:

Dots & Dashes: . - Spaces: normal spaces

IMPORTANT: All Morse code block must have / between each character and space

Unsupported characters will display as # in the output
